시간복잡도
	알고리즘의 수행 시간을 분석할 때 시간 복잡도를 사용
	-최선의 경우 (Best Case)
		빅 오메가 표기법 사용
		최선의 시나리오로 최소 이만한 시간이 걸림

	-최악의 경우 (Worst Case)
		빅 오 표기법 사용
		최악의 시나리오로 아무리 오래 걸려도 이 시간보다 덜 걸림

	-평균적인 경우 (Average Case)
		빅 세타 표기법 사용
		평균 시간을 나타냄

공간 복잡도(Space Complexity)
	공간 복잡도는 알고리즘에서 사용하는 메모리 양을 나타냄

스택
	가장 먼저 삽입된 자료가 가장 먼저 삭제되는 자료구조(LIFO)
	-연산
		push : 데이터 삽입
		pop  : 데이터 삭제
